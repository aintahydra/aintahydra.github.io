Android Studio

Install Android Studio

  • Download the recent version of Android Studio (https://developer.android.com/)
  • Unpack the tarball and set paths 
    tar xvfz android-studio-2022.1.1.21-linux.tar.gz
sudo mv android-studio /usr/local
    and add the below to the bash_profile 
    export ANDROID_HOME=$HOME/Android/Sdk
export PATH=$PATH:/usr/local/android-studio/bin:$ANDROID_HOME/tools
  • install adb (apt install adb)
  • Run studio.sh

Add an android virtual device (AVD)

  • Tools > Device Manager

  • Create a Device (e.g., Pixel 2; Nougat; API Level 24; Target 7.0; NON-GoogleAPIs)

  • Listing avds; installed devices are located at (~/.android/avd)

    $ emulator -list-avds
Pixel_2_API_24_No_GPlay

$ ls ~/.android/avd/
Pixel_2_API_24_No_GPlay.avd  Pixel_2_API_24_No_GPlay.ini

  • Starting avd

    $ cd ~/Android/Sdk/tools
$ emulator -avd Pixel_2_API_24_No_GPlay
